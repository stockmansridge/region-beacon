# MVP Scope — Event Passport SaaS

## Version 1 — In Scope

1. Visitor registration
2. QR check-ins
3. Passport progress tracking
4. Admin dashboard
5. Event creation
6. Venue management
7. QR code generation
8. QR poster download
9. Basic analytics
10. CSV export
11. White-label event branding

## Version 1 — Out of Scope

- Native iOS/Android app
- Maps integration
- Offers / rewards engine
- Browser push notifications
- Social sharing features
- Team challenges
- CRM integrations
- Automated winner selection
- Advanced fraud scoring

## Scope Change Policy

Adding an out-of-scope item to Version 1 requires explicit approval. Removing an in-scope item also requires explicit approval. Do not silently expand or shrink the MVP boundary.
